# PickerTextField
UIPickerViewで入力するTextField
## 使い方

```swift
let ptf = PickerTextField(frame: CGRectMake(100, 100, 100, 30))
ptf.setup(["Ei!", "Fire!", "Icestorm!", "Diacute!", "Braindumbed!", "Jugem!", "Bayoen!"])
```
